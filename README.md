# 🕒 Tab Lifecycle Timer

> **Автоматическое закрытие неактивных вкладок** с панелью управления, белым списком, корзиной и защитой медиа.

## 📋 Возможности

| Функция | Описание |
|---|---|
| ⏱ **Авто-закрытие** | Закрывает вкладки, простоявшие в фоне заданное время |
| 🎯 **Локальный иммунитет** | Защити конкретную вкладку через попап — даже без закрепления |
| 🔒 **Белый список** | Домены, которые расширение никогда не закроет |
| 🗑 **Корзина** | Последние 50 закрытых вкладок — с возможностью восстановления |
| 🧠 **Защита медиа** | Видео/аудио на паузе не дадут закрыть вкладку |
| 🎬 **Сохранение таймкода** | YouTube, Vimeo, Twitch — восстановление с того же места |
| ⏸ **Глобальная пауза** | Один тумблер — временно отключить всю автоматику |
| 📊 **Аналитика RAM** | Счётчик сэкономленной оперативной памяти |
| 🛡 **Защита Dashboard** | Панель управления не закроется сама собой |
| 🔍 **Живой поиск** | Фильтрация корзины и защищённых вкладок на лету |

## 🚀 Установка

### Chromium (Chrome, Edge, Brave)

1. Открой `chrome://extensions/`
2. Включи **Режим разработчика** (справа вверху)
3. Нажми **«Загрузить распакованное расширение»**
4. Выбери папку с проектом

### Firefox

1. Открой `about:debugging#/runtime/this-firefox`
2. Нажми **«Загрузить временное дополнение…»**
3. Выбери любой файл из папки (`manifest.json`)

> При первом запуске в Firefox появится баннер с запросом доступа к сайтам — нажми **«Разрешить доступ»**.

### Сборка из исходников

```bash
python build.py
# → dist/chromium/   (Service Worker)
# → dist/firefox/    (Event Pages)
```

## ⚙️ Как это работает

```
Вкладка создана/активирована
        │
        ▼
   Запущен таймер (N мин)
        │
        ▼
   Таймер сработал ───┬── Вкладка в фокусе? ──→ Сброс таймера
                      │
                      ├── Закреплена/звучит? ──→ Сброс таймера
                      │
                      ├── В белом списке? ────→ Сброс таймера
                      │
                      ├── Видео на паузе? ────→ Сброс таймера
                      │
                      └── Всё чисто ──────────→ Закрыть + Лог в корзину
```

## 🛠 Файлы конфигурации

| Ключ | Тип | По умолчанию | Описание |
|---|---|---|---|
| `timeoutMinutes` | `number` | `10` | Таймаут до закрытия (мин) |
| `timerEnabled` | `boolean` | `true` | Глобальная пауза |
| `whiteList` | `string[]` | `[]` | Домены белого списка |
| `protectedTabIds` | `number[]` | `[]` | ID вкладок с иммунитетом |
| `protectDashboard` | `boolean` | `true` | Защита панели управления |
| `trashBin` | `object[]` | `[]` | Корзина (до 50 записей) |
| `savedRamMb` | `number` | `0` | Счётчик сэкономленной RAM |

## 🔐 Безопасность

- ❌ Нет доступа к кукам, истории, паролям
- ❌ Нет телеметрии и внешних запросов
- ✅ Все данные хранятся локально в `chrome.storage.local`
- ✅ Защита от XSS: рендер через `createTextNode`, не через `innerHTML`
- ✅ Защита от Race Condition: очередь Promise для storage
